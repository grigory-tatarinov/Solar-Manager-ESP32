## Solar-Manager-ESP32
 Connection:&nbsp;
ESP32   - 1602 LCD + I2C Converter
* GND		-	 GND &nbsp;
* VIN  	-	 VCC &nbsp;
* D21  	-	 SDA &nbsp;
* D22  	-	 SCL &nbsp;


ESP32   - HW-040 Encoder
* GND		  -	 GND(1pin)  &nbsp;
* VIN  	  -	 5V(2pin)  &nbsp;
* D15  	  -	 SW(3pin)  &nbsp;
* D5	 	  -	 DT(4pin)  &nbsp;
* D4  	  -	 CLK(5pin)  &nbsp;
* You can change these two pin D4 Ð¸ D5 and than increase direction will also changed  &nbsp;


ESP32   - 1ch 30A Relay
* GND		  -	 DC-(1pin)  &nbsp;
* VIN  	  -	 DC+(2pin)  &nbsp;
* D13  	  -	 IN(3pin)  &nbsp;
